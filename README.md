# Carrera

Videojuego en Unity 🎮

Desarrollé un videojuego en Unity utilizando C#, en el cual implementé conceptos avanzados de programación orientada a objetos como herencia, polimorfismo y colisiones. En el juego, el personaje principal recorre una ciudad atrapando monedas mientras escapa de un enemigo, ofreciendo una experiencia interactiva y dinámica. El proyecto incluye el diseño de múltiples escenas y una lógica de juego estructurada para mantener el flujo y la dificultad del desafío.
